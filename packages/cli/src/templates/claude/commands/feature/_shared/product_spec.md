## 快速识别指南

### 如何识别产品类型？

**步骤**：

1. 在需求分析报告（`testcase_analysis.md`）中搜索产品名称
2. 使用上表中的关键词进行匹配
3. 确认对应的产品类型

**示例**：

```
需求分析报告中提到："本项目为A9解码器增加水印功能"

识别过程：
1. 搜索关键词"A9"
2. 匹配到"A9系列"属于"屏控产品"
3. 确认产品类型：屏控产品（PK）
4. 对应Checklist：testcase_checklist_PK.md
```

---

## 使用建议

1. **产品类型识别**：优先使用本文件进行快速匹配
2. **详细规格查询**：对于特定产品的详细测试点，参考对应的Checklist文件
3. **组合产品**：如果一个项目涉及多个产品类型，需分别使用对应的Checklist进行验证
